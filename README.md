# Guia Studeo

Projeto simples desenvolvido como parte dos meus estudos iniciais em **HTML** e **CSS**.

O site tem como objetivo colocar em prática meus conhecimentos em desenvolvimento web, criando uma página com orientações básicas sobre a metodologia dos cursos **EAD da Unicesumar**.

[🔗 Acesse o site online](https://fabraoliveira.github.io/guia-studeo/)

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
